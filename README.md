📊 Budget Tracker – Finance Management System

A Django-based personal finance management system that helps users track their income, expenses, and overall balance with detailed analytics and visual charts.

🚀 Features
👤 User-based login system
💰 Add and manage earnings
💸 Add and manage expenses
🏷️ Expense categorization system
📍 Location tracking for entries (manual + map selection)
📅 Date-based filtering of records
📊 Category-wise expense breakdown
📈 Visual analytics using charts (Chart.js)
🧾 View, edit, and delete transactions
🏦 Salary + additional income support
📉 Profit/Loss balance tracking
